Make commands:

make q3: compiles and runs 2 files one by one that is binary search tree part and height-balanced binary search tree that is AVL.
make q4: complies and runs program for ques4

All question take console input and give console output and work self-explanatory.
Working is demonstrated by taking output by from user and working on it.
